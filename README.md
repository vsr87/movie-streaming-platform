# movie-streaming-platform
Repository created for the Software and Systems Engineering (ESS) course project. 

## 🛠️ Desenvolvimento com DevContainer
Para garantir que todos do grupo usem a mesma versão do Node.js 24 e as extensões necessárias, serão utilizados Dev Containers. Isso evita conflitos entre versões diferentes das dependências e configurações do projeto.

### Pré-requisitos
- Docker instalado na máquina.
- Extensão Dev Containers (da Microsoft) instalada no seu VSCode.

### Como rodar
1. Abra a pasta raiz do projeto no seu VSCode.
2. Um aviso aparecerá no canto inferior direito perguntando se deseja reabrir a pasta em um container. Clique em ```Reopen in Container```.
3. Caso o aviso não apareça: Pressione ```Ctrl + Shift + P``` e digite ```Dev Containers: Reopen in Container```. O VSCode irá baixar a imagem e configurar o ambiente automaticamente.
4. Quando a barra de status no canto inferior esquerdo mostrar Dev Container: Node.js 24..., o ambiente está pronto.
